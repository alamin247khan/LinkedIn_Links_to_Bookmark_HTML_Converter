✅ Usage Steps:
1. Prepare the Input:

You can use:
A text file with one LinkedIn URL per line.
A CSV file with LinkedIn URLs in a specific column (default is "url").
The clipboard (you can copy LinkedIn URLs directly).

2. Run the Script:
Navigate to the folder where the script is located and use the following command format:

bash

Edit

3. Optional Arguments:
--delay [seconds]: Set a custom delay between requests.
--skip-errors: Continue processing even if some URLs fail.
--show-config: Show the default folder configuration and exit.
--config [config_file]: Use a custom JSON configuration for folder categorization.

4. Result:
The generated HTML file (e.g., bookmarks.html) will contain categorized bookmarks.
You can import this file directly into browsers like Chrome or Firefox to access the organized LinkedIn profiles.
